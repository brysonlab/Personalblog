# Personalblog

A Django-based blog application built with Python and Django framework.

## 🚀 Overview

Personalblog is a web application that allows users to:

- Create and publish blog posts
- View post details
- Manage content via Django admin
- Organize content using Django models and templates

This project demonstrates core Django concepts including:

- Models
- Views
- Templates
- URL routing
- Admin configuration
- Database migrations

---

## 🛠 Tech Stack

- Python 3.x
- Django
- SQLite (default development database)
- HTML (Django Templates)
